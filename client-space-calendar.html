Importer un fichier Excel / CSV

Dépose un fichier avec ta base de contenus : le site lit les lignes, affiche un aperçu, puis les ajoute au calendrier. Les colonnes sont reconnues automatiquement même si les titres ne sont pas exactement identiques.

Fichier accepté : .xlsx, .xls ou .csv Choisis le fichier, vérifie l’aperçu, puis clique sur “Importer dans le calendrier”.
Colonnes conseillées :

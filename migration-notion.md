# Plan de migration depuis Notion

Comme le contenu Notion n'a pas pu être récupéré automatiquement ici, voici la méthode de migration rapide :

## 1. Structure conseillée

Chaque page Notion devient un module du site :

- Titre de la page → Titre du module
- Objectif → Objectif pédagogique
- Paragraphes / consignes → Contenu
- Liens PDF, Drive, Canva, vidéo → Ressources
- Travail demandé → Livrable associé

## 2. Format recommandé pour les ressources

Dans l'espace Gestion, champ `Ressources`, ajoute une ressource par ligne :

```txt
Nom du PDF | https://lien-vers-le-pdf.com
Replay séance 1 | https://lien-video.com
Template questionnaire | https://lien-drive.com
```

## 3. Modules déjà préparés

Le site contient déjà une base pour :

1. Introduction à l’étude de marché
2. Définir son offre et ses hypothèses
3. Identifier la cible et construire un persona
4. Analyser la concurrence
5. Construire un questionnaire d’enquête
6. Synthèse et décision finale

## 4. Ce qu'il faut compléter

- Les vrais liens des vidéos ou lives.
- Les supports PDF ou Google Drive.
- Les dates limites.
- Le lien du groupe WhatsApp.
- Les consignes précises des livrables.
